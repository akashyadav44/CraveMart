# 🍜 CraveMart - Full Stack Food Ordering Platform

CraveMart is a modern full-stack food ordering and e-commerce web application that allows users to browse food items, create accounts, log in, manage carts, and place orders seamlessly.

## 🚀 Live Demo

### Frontend
https://crave-mart-nine.vercel.app

### Backend API
https://cravemart.onrender.com

---

## ✨ Features

- User Registration & Login
- JWT Authentication
- Browse Food Products
- Shopping Cart Functionality
- Order Management
- Responsive Design
- RESTful API Architecture
- MongoDB Database Integration
- Secure Password Hashing
- Cross-Origin Resource Sharing (CORS) Support

---

## 🛠️ Tech Stack

### Frontend
- HTML5
- CSS3
- JavaScript (ES6)

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas
- Mongoose

### Authentication
- JWT (JSON Web Token)
- bcryptjs

### Deployment
- Frontend: Vercel
- Backend: Render
- Database: MongoDB Atlas
